Drivers

MPLAB Harmony Graphics Suite provides capability to create board specific c-language graphics drivers for external hardware communication. The suite supports code generation for following external hardware categories: 

 

  • Display Controller Driver - MHGS generates driver code for a wide range of displays, with a focus on QVGA to WVGA LCDs. OLED and multi-layer display controllers are also supported. Drivers are generated for SPI and parallel interfaces.
  • GPU Driver - MHGS generates driver code for the PIC32MZ DA 2DGPU and the SAM GFX2D peripheral graphics accelerator.
  • Touch Screen Driver - MHGS generates code through its generic driver component for Microchip maXTouch controller. The genertic driver can bbe for single, two finger-touch, and gesture support for 2D maXTouch Touch Screen controllers.
